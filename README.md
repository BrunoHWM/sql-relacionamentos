# SQL Relacionamentos

## CONSULTA SQL 

Este repositório contém os exercícios de **SQL** desenvolvidos durante as aulas da turma **TDS26-1N**.

O projeto reúne exemplos de:

* Criação de tabelas (`CREATE TABLE`);
* Inserção de dados (`INSERT INTO`);
* Consultas com `SELECT`;
* Filtros utilizando `WHERE`;
* Ordenação de resultados com `ORDER BY`;
* Pesquisas com `LIKE`;
* Relacionamentos entre tabelas por meio de chaves estrangeiras.

O objetivo é praticar os principais conceitos de **Banco de Dados Relacional**, desenvolvendo consultas SQL e aplicando boas práticas de organização e estruturação de dados.
