# Exercícios SQL - TDS26-1N  Repositório contendo exercícios de SQL desenvolvidos durante as aulas da turma **TDS26-1N**. O projeto inclui criação de tabelas, inserção de dados e consultas utilizando comandos como `SELECT`, `WHERE`, `ORDER BY`, `LIKE` e relacionamentos entre tabelas, com o objetivo de praticar conceitos de Banco de Dados Relacional
